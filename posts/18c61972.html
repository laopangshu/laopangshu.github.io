python自学笔记

发表于2023-07-11|更新于2023-07-29|学习python
|字数总计:955|阅读时长:3分钟|阅读量:

自学

​ 第一本书籍——python编程从入门到实践2016

安装

按住shift+鼠标右键，在弹出对话框中输入python，在商城中下载python，开始使用python

内容

变量

使用变量使得print代表的函数内容较少

message = "Hello Python world!"

print(message)

message = "Hello Python Crash Course world!"

print(message)

输出的为

Hello Python world!

Hello Python Crash Course world!

变量命名的注意事项

​ 使用变量命名时，不能命名数字在前，也不可有空格出现，可用_代替空格。

​ 使用变量时，容易输出时打错或打少字母个数，不用太规范，变量可随意字母组合，不是固定的英文名称，若出现打错，会出现报错无定义此变量。

字符串

定义

​ 就是一系列的字符，在python通过单引号或双引号进行表示，在单引号和双引号的中间，则为字符串内容。

使用方法修改字符串的大小写

name = "ada lovelace"

print(name.title())

以上输出内容为

Ada Lovelace

！！！name.title()仅仅为首字母大写，name.upper()为全部大写，name.lower()为全部小写。

合并(拼接)字符串

first_name = "ada"

last_name = "lovelace"

full_name = first_name + " " + last_name

print(full_name)

以下为多种组合

first_name = "ada"

last_name = "lovelace"

full_name = first_name + " " + last_name

print("Hello, " + full_name.title() + "!")

输出为Hello, Ada Lovelace!

还可以将其设为变量，直接print变量即可

first_name = "ada"

last_name = "lovelace"

full_name = first_name + " " + last_name

message = "Hello, " + full_name.title() + "!"

print(message)

输出结果也为上面结果

使用制表符或换行符来添加空白

print输出可有\n ,\t, \n表示换行，\t表示添加制表符。

删除空白

``favorite_language = ‘python ‘`

**favorite_language**

'python '

favorite_language.rstrip()

'python'

rstrip()—表示右边的空格删除，lstrip表示左边的空格删除，strip表示两边都删

使用字符串时避免语法错误

不要在‘ 中添加‘，或在’‘中添加“

数字

1、整数操作 +- / （*表示幂次）

2、浮点数，也有出现0.2 + 0.1 = 0.3000000004无确定后面几位的情况，属于正常，python2中会出现3/2=1的情况，若需要显示1.5，只需要把一个数精确到小数点后一位即可。

3、 使用str（）将数字或其他转化为字符串，即可输出，str()函数就是将非字符串值转化为字符串，

注释

注释用#后面即可注释

Python之禅

输入import this 即可查看

列表

列表用[]表示

列表中的每一个元素相隔通过，隔开

访问列表元素

访问列表元素只需要在print(内填入)列表名称[]并在中括号中填入对应位置数字

索引从0开始不是从1开始

索引中数字输入-1，意思为倒数第一个，-2为倒数第二个

使用列表中的各个值

可像使用其他变量一样使用列表中的各个值，然后将其拼接为一个句子

修改、添加和删除元素

修改值可通过以下式子

s = [‘f’,’g’,’fa’]

s[0] = ‘a’

print(s)

s[什么] = 什么

添加元素可以通过以下式子

s = [‘f’,’g’,’fa’]

s.append(‘baf’)

print(s)

s.append(‘什么’)

在列表中插入元素可通过以下式子

s = [‘f’,’g’,’fa’]

s.insert(1,’c’)

print(s)

s.insert(位置，’什么‘)

在列表中删除元素可通过以下式子

  • 使用del语句删除元素

del s[0]

  • 使用pop（）删除元素—弹出，之后还要用
  • 弹出列表中的任一元素，在pop()中的括号中添加数字即可
  • 根据值删除元素s.remove(‘元素’)

注意：remove删除的只是第一个出现的值，若还有其他值，可以通过循环删除
